=========================================
-- BD2 - Taller 1: Diseño de Consultas Avanzadas SQL - Sistema de Biblioteca
=========================================
Este repositorio contiene la resolución del Taller individual 1 para la asignatura de Bases de Datos II. El objetivo es demostrar el dominio de consultas complejas en SQL, incluyendo subconsultas, uniones, funciones especializadas y manejo de fechas sobre un modelo de datos de una biblioteca.

=========================================
--Descripción del Proyecto
=========================================
El taller se basa en un sistema de gestión de biblioteca que permite administrar libros, usuarios, préstamos y reservas. Se implementaron diversas consultas para extraer información estratégica y realizar análisis de datos sobre el comportamiento de los usuarios y el inventario de libros.

=========================================
-- Tecnologías utilizadas
=========================================
Lenguaje: SQL (Standard Query Language)
Motor de Base de Datos: MySQL
